\noindent Recall Euler's Formula: $e^{ix} = \cos x + i\sin x$. Let $z = \cos x + i\sin x$.
\begin{align*}
\frac{dz}{dx} &= -\sin x + i\cos x = i(i\sin x + \cos x) \\
&= i(\cos x + i\sin x)\\
&= iz\\
\implies \frac{dz}{z} &= i \, dx, \,\, \int \frac{1}{z} dz = \int i \, dx\\
\ln z &= ix \implies z = e^{ix} = \cos x + i\sin x
\end{align*}
